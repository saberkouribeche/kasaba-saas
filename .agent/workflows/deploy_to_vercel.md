# نشر التحديثات على Vercel يدوياً

يمكنك نشر التحديثات بسهولة باستخدام واجهة الأوامر (Command Line).

## 1. نشر نسخة تجريبية (Preview)
لنشر نسخة للمراجعة فقط (لا تؤثر على الرابط الرئيسي):
```bash
npx vercel
```
- سيطلب منك التأكيد، اضغط `Enter` لجميع الأسئلة.
- سيعطيك رابطاً خاصاً بهذه النسخة.

## 2. نشر نسخة حقيقية (Production)
لنشر التعديلات على الرابط الحي (الرئيسي) ليراها الجميع:
```bash
npx vercel --prod
```
- هذا الأمر يقوم ببناء المشروع ورفعه فوراً.

## 3. النشر بالقوة (في حال وجود مشاكل)
إذا شعرت أن التحديثات لا تظهر بسبب "الذاكرة المؤقتة" (Cache)، استخدم هذا الأمر لإجبار النظام على إعادة البناء من الصفر:
```bash
npx vercel --prod --force
```
